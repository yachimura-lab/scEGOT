# scEGOT

single cell trajectory inference framework based on Entropic Gaussian mixture Optimal Transport

## Instalattion

```
$ pip install scegot
```

## Requirements

- Python3
- pandas
- anndata
- GraphViz

  Click [here](https://graphviz.org/) to learn how to install GraphViz (official documentation).
